# Problem 011

> BOJ 1874 스택 수열
> <br/>
> [링크](https://www.acmicpc.net/problem/1874)

## 문제

주어진 수열이 완성되는지 체크하는 문제다.

## 풀이

이 문제의 핵심 알고리즘은 `Stack`의 작동원리인 `FILO`을 이용하는 것이다.

## 해결 전략

- 오름차순으로 숫자 push
- 목표 값에 도달 시 pop
- 수열 불가능 조건 검사
- 모든 연산을 수행하여 주어진 수열이 완성되면, 각 연산(+, -)을 한 줄씩 출력

스택의 `push`와 `pop` 연산을 각각 한 번씩 수행하므로, 시간 복잡도는 `O(N)`이다.
