# Problem 027

> BOJ 2178 미로탐색
> <br/>
> [링크](https://www.acmicpc.net/problem/2178)

## 문제

`N × M` 크기의 배열로 표현되는 미로가 주어질 때, `(1, 1)`에서 출발하여 `(N, M)` 위치로 이동할 때 지나야 하는 최소 칸 수를 구하는 문제다.
1은 이동할 수 있는 칸을 의미하고, 0은 이동할 수 없는 칸을 의미한다. 칸을 셀 때는 시작 위치와 도착 위치도 포함한다.

## 풀이

이 문제는 `BFS`를 사용하여 최단 경로를 찾는 전형적인 문제다. `BFS`는 시작점에서부터 인접한 노드를 탐색하며 점진적으로 거리를 늘려가므로, 처음 도달한 목적지까지의 경로가 곧 최단 경로가 된다.

## 해결 전략

- 입력받은 미로 지도를 2차원 배열로 저장하고, 각 칸의 방문 여부를 기록하기 위해 `visited` 배열을 사용한다.
- 큐를 이용하여 `BFS`를 수행한다. `(0, 0)` 위치에서 시작하여 인접한 4방향(상, 하, 좌, 우)으로 이동하며, 이동할 수 있는 칸에 현재까지의 이동 횟수를 기록한다.
- 이동할 때마다 이전 칸까지의 이동 횟수에 1을 더해 현재 칸의 이동 횟수로 기록한다. BFS 특성상 첫 번째로 목적지에 도달하면 해당 경로가 최단 경로가 된다.
- `BFS`가 종료된 후 목적지 좌표 `(N-1, M-1)`에 기록된 값이 최소 이동 횟수이므로, 이를 출력한다.

`BFS`를 사용해 풀이했으므로 시간복잡도는 `O(N * M)` 이다.
